Установка Hyprland на «голую» систему (без рабочего стола) в CachyOS — отличный выбор. CachyOS основан на Arch Linux и имеет собственные оптимизированные репозитории, поэтому процесс очень похож на классический Arch, но система будет работать быстрее.

Ниже приведено пошаговое руководство по превращению консольной CachyOS в полноценную систему с Hyprland.

### Шаг 1: Обновление системы
Перед началом установки убедитесь, что система полностью обновлена:
```bash
sudo pacman -Syu
```

### Шаг 2: Установка драйверов видеокарты (Критически важно для Wayland)
Hyprland работает на протоколе Wayland. То, как он будет работать, зависит от вашей видеокарты.

*   **Для AMD / Intel:**
    Драйверы встроены в ядро, но нужны пакеты Vulkan и OpenGL:
    ```bash
    sudo pacman -S mesa vulkan-radeon lib32-vulkan-radeon lib32-mesa # Для AMD
    # ИЛИ
    sudo pacman -S mesa vulkan-intel lib32-vulkan-intel lib32-mesa   # Для Intel
    ```

*   **Для NVIDIA (Требует внимания):**
    Wayland на Nvidia работает хорошо на современных драйверах, но требует установки проприетарных драйверов и модулей ядра. В CachyOS есть удобная утилита для этого:
    ```bash
    sudo chwd -a pci nonfree 0300
    ```
    *(Или вручную: `sudo pacman -S nvidia-dkms nvidia-utils egl-wayland`)*

### Шаг 3: Установка базовых компонентов Hyprland
Установим сам композитор, порталы (для демонстрации экрана и работы Flatpak) и поддержку Qt-приложений:
```bash
sudo pacman -S hyprland xdg-desktop-portal-hyprland qt5-wayland qt6-wayland polkit-kde-agent
```

### Шаг 4: Установка необходимых программ (Окружение)
Hyprland — это только оконный менеджер. Вам понадобятся терминал, панель, меню запуска приложений и т.д.
```bash
sudo pacman -S kitty rofi-wayland waybar hyprpaper mako thunar network-manager-applet
```
*   `kitty` — терминал (по умолчанию прописан в конфиге Hyprland).
*   `rofi-wayland` — меню запуска приложений (лаунчер).
*   `waybar` — статус-бар (панель).
*   `hyprpaper` — установка обоев.
*   `mako` — демон уведомлений.
*   `thunar` — файловый менеджер.
*   `network-manager-applet` — иконка Wi-Fi/Сети в трее.

### Шаг 5: Аудио и Bluetooth
CachyOS обычно использует Pipewire по умолчанию, но на всякий случай установим его и утилиты для звука и Bluetooth:
```bash
sudo pacman -S pipewire pipewire-pulse pipewire-alsa wireplumber pavucontrol bluez bluez-utils blueman
sudo systemctl enable bluetooth
```

### Шаг 6: Шрифты (Важно для отображения иконок в Waybar)
Чтобы иконки в статус-баре и терминале отображались корректно, нужны Nerd Fonts:
```bash
sudo pacman -S ttf-nerd-fonts-symbols-common ttf-jetbrains-mono-nerd noto-fonts-emoji
```

### Шаг 7: Менеджер входа (Экран приветствия)
Чтобы не запускать Hyprland каждый раз из консоли командами, установим дисплейный менеджер **SDDM** (он лучше всего работает с Wayland):
```bash
sudo pacman -S sddm
sudo systemctl enable sddm
```

### Шаг 8: Базовая настройка перед первым запуском
Создайте директорию для конфигов и скопируйте стандартный конфиг Hyprland:
```bash
mkdir -p ~/.config/hypr
cp /usr/share/hyprland/hyprland.conf ~/.config/hypr/
```

*(Опционально)* Если у вас **NVIDIA**, вам нужно добавить параметры среды в конфиг. Откройте его:
```bash
nano ~/.config/hypr/hyprland.conf
```
И добавьте в самое начало файла следующие строки:
```text
env = LIBVA_DRIVER_NAME,nvidia
env = XDG_SESSION_TYPE,wayland
env = GBM_BACKEND,nvidia-drm
env = __GLX_VENDOR_LIBRARY_NAME,nvidia
env = WLR_NO_HARDWARE_CURSORS,1
```
*Сохраните (Ctrl+O, Enter) и выйдите (Ctrl+X).*

### Шаг 9: Запуск
Теперь всё готово. Перезагрузите компьютер:
```bash
reboot
```

После перезагрузки вы увидите экран входа SDDM. Выберите сессию **Hyprland** (обычно в левом нижнем или верхнем углу) и введите свой пароль.

### Что делать после первого входа?
1. Вы увидите предупреждение сверху экрана, что используется стандартный конфиг. Нажмите **Super + Q** (или **Windows + Q**), чтобы открыть терминал `kitty`.
2. В терминале откройте файл конфигурации для редактирования под себя:
   ```bash
   nano ~/.config/hypr/hyprland.conf
   ```
3. Найдите строку `autogenerated = 1` и удалите её (или закомментируйте, добавив `#` в начале), чтобы убрать желтое предупреждение.
4. В разделе `autostart` добавьте автозапуск панели и обоев:
   ```text
   exec-once = waybar &
   exec-once = hyprpaper &
   exec-once = nm-applet &
   ```

**Совет:** CachyOS имеет пакет готовых настроек для Hyprland (dotfiles). Если вы не хотите настраивать всё с нуля, вы можете установить их пакет:
```bash
sudo pacman -S cachyos-hyprland-settings
```
И скопировать их в свою папку `~/.config/`, перезаписав стандартные файлы.
